# Identifikation-von-Risikokorridoren-von-Wildschweinen-mittels-Least-Cost-Path-Modellierung-LCP-
Identification of wild boar risk corridors in Miltenberg using Least-Cost Path modeling. Integrates geospatial data and R workflows to map seasonal habitats, movement routes to water and farmland, and support African Swine Fever prevention strategies.
